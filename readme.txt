== Description ==

Quick Search And Only One Category Posting support quickly selecting category via search box when there are many categories. Besides, you can use radio button instead of checkbox when you want clients to choose 1 category only. 
Quick Search And Only One Category Posting supports all custom post types and custom taxonomy activated in theme and plugin.

== Installation ==

Activate the plugin. Go Settings -> QSOOCP. Select Post Type, Taxonomy... Done !
